# 农财通 - 智慧农业金融服务平台

农财通是一个专注于农业金融科技的创新平台，为农业经营者提供智能化的财务管理和补贴申请服务。

## 主要功能

### 1. 农业补贴管理
- 补贴政策查询
- 在线补贴申请
- 申请进度跟踪
- 补贴发放记录

### 2. 财务记账管理
- 日常收支记录
- 农资采购管理
- 销售收入统计
- 财务报表生成

### 3. 数据分析报告
- 收益分析报告
- 成本核算分析
- 市场行情预测
- 经营状况评估

### 4. 农业保险服务
- 保险产品推荐
- 在线投保服务
- 理赔进度查询
- 风险评估报告

## 技术栈

- **前端框架**: Next.js 13
- **UI 组件**: Radix UI
- **样式方案**: Tailwind CSS
- **状态管理**: React Hooks
- **轮播组件**: Embla Carousel
- **图标**: Lucide Icons

## 开发环境

1. 安装依赖
npm install
2. 启动开发服务器
npm run dev
3. 构建生产版本
npm run build
4. 启动生产服务器
npm run start
